1. 链接的定义格式<a>:

<a href="url">Link text</a> 

    <a href="http://www.jasonkay.club/">Visit jasonkay</a>

注:
  1. "链接文本" 不必一定是文本。图片或其他 HTML 元素都可以成为链接。
  2. href 属性规定链接的目标。
  3. 开始标签和结束标签之间的文字被作为超级链接来显示。

2. 链接的target属性:

    <a href="http://www.jasonkayzk666.club/" target="_blank">Visit Jasonaky!</a>

注:
  使用 Target 属性，你可以定义被链接的文档在何处显示。
  "_blank"会在新窗口打开文档.

3. 链接的name属性:

  name 属性规定锚（anchor）的名称。
  您可以使用 name 属性创建 HTML 页面中的书签。
  书签不会以任何特殊方式显示，它对读者是不可见的。
  当使用命名锚（named anchors）:
  我们可以创建直接跳至该命名锚（比如页面中某个小节）的链接，这样使用者就无需不停地滚动页面来寻找他们需要的信息了. 

例:
    首先，我们在 HTML 文档中对锚进行命名（创建一个书签）：
    <a name="tips">基本的注意事项 - 有用的提示</a>
    
    可以在其他页面中或在同一个文档中创建指向该锚的链接：
    
    <a href="#tips">有用的提示</a>
    <a href="http://www.jasonkayzk666.club#tips">有用的提示</a>

注:
  请始终将正斜杠添加到子文件夹。
  假如这样书写链接：href="http://www.jasonkayzk666.club/html"，就会向服务器产生两次 HTTP 请求。这是因为服务器会添加正斜杠到这个地址，然后创建一个新的请求，就像这样：href="http://www.jasonkayzk666.club/html/"。
  假如浏览器找不到已定义的命名锚，那么就会定位到文档的顶端。不会有错误发生。
