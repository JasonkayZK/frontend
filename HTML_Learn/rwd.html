1. 响应式Web设计(RWD):

  • RWD 指的是响应式 Web 设计（Responsive Web Design）
  • RWD 能够以可变尺寸传递网页
  • RWD 对于平板和移动设备是必需的

2. 创建一个自己的RWD:

Jasonkay Demo
