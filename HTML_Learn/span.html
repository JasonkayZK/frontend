1. span元素:

    HTML <span> 元素是内联元素，可用作文本的容器。

    <span> 元素也没有特定的含义。

    当与 CSS 一同使用时，<span> 元素可用于为部分文本设置样式属性。

2. span与class连用:

My Important Heading. 

Code:
    <!DOCTYPE html>
    <html>
    <head>
    <style>
      span.red {color:red;}
    </style>
    </head>
    <body>

    <h1>My <span class="red">Important</span> Heading</h1>

    </body>
    </html>
