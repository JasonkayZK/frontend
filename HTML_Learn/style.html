1. 样式<style>代替了原本的一些HTML标签

如: <center>, <font>, <basefont>, <s>, <strike>, <u>, <align>, <color>, <bgcolor>
  请使用style或CSS来代替!


1. <style≡"background-color: ***" >来设置背景:

源代码:
    <html>
    <body style="background-color:yellow">
    <h2 style="background-color:red">This is a heading</h2>
    <p style="background-color:green">This is a paragraph.</p>
    </body>
    </html>

This is a heading

This is a paragraph.


2. <style≡"font-family: ***">来设置字体:

源代码:
    <html>
    
    <body>
    <h1 style="font-family:verdana">A heading</h1>
    <p style="font-family:arial;color:red;font-size:20px;">A paragraph.</p>
    </body>

    </html>

A heading

A paragraph.


3. <style≡"text-align: ***">来设置文本对其方式:

源代码:
    <html>

    <body>
    <h1 style="text-align:center">This is a heading</h1>
    <p>The heading above is aligned to the center of this page.</p>
    </body>

    </html>

This is a heading

The heading above is aligned to the center if this paragraph.
