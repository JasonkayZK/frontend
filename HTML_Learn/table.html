1. 表格的定义<table border="***">:

row1, cell1 row1, cell2 row1, cell3
row2, cell1 row2, cell2   
源代码:
       <table border="1">
        <tr>
            <td>row1, cell1</td> 
            <td>row1, cell2</td>
            <td>row1, cell3</td>
        </tr>
        <tr>
            <td>row2, cell1</td> 
            <td>row2, cell2</td>
            <td> </td>
        </tr>
       </table>
注:
  表格由 <table> 标签来定义。
  每个表格均有若干行（由 <tr> 标签定义），
  每行被分割为若干单元格（由 <td> 标签定义）。
  字母 td 指表格数据（table data），即数据单元格的内容。
  数据单元格可以包含文本、图片、列表、段落、表单、水平线、表格等等。

2. 表格的边框 <table border&equiv"***">:

  border = 0 表示无边框(默认无边框)

            </tr>
            
        </table>

4. 表格中的空白单元格:

注:
  在一些浏览器中，没有内容的表格单元显示得不太好。
  如果某个单元格是空的（没有内容），浏览器可能无法显示出这个单元格的边框。

  这个空的单元格的边框没有被显示出来。
  为了避免这种情况，在空单元格中添加一个空格占位符，就可以将边框显示出来。

row1, cell1 row1, cell2
  row2, cell2

9. 单元格边距和间距(cellpadding/cellspacing):

Normal:

First Row
Second Row

Cellpadding:

First Row
Second Row

Cellspacing:

First Row
Second Row 
源代码:
        <h4>Cellpadding:</h4>
        <table border="1" cellpadding="10">
            <tr>
                <td>First</td>
                <td>Row</td>
            </tr>
            <tr>
                <td>Second</td>
                <td>Row</td>
            </tr>
        </table>
        
        <h4>Cellspacing:</h4>        
        <table border="1" cellspacing="10">
            <tr>
                <td>First</td>
                <td>Row</td>
            </tr>
            <tr>
                <td>Second</td>
                <td>Row</td>
            </tr>
        </table>
